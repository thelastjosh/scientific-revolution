# Feature Plan: Sustainability & Recognition

## 1. The "Drumbeat": Global Campaigns (Epochs)
**Goal:** Create a collective sense of urgency and progress.
**Concept:** Instead of an endless stream of isolated tasks, organize work into "Epochs" or "Operations".
**Feature:**
- **Global Status Board:** A persistent header or dashboard widget showing the current "Operation" (e.g., "Operation: Sector 7 Integrity").
- **Collective Progress:** A global progress bar that fills up as everyone completes tasks.
- **The "Beat":** Weekly or daily targets. If the community misses the target, the "System Status" degrades (visual feedback like increased noise/glitches).

## 2. Recognition: The "Service Dossier"
**Goal:** Provide proof of skill without gamifying intrinsic motivation away with generic badges.
**Concept:** A portable, professional record of "Artifacts" created.
**Feature:**
- **The Dossier:** A public profile page that looks like a classified personnel file.
- **Artifact Registry:** Instead of "Badges", users earn "Artifacts" (e.g., "Verified Dataset #8821", "Algorithm Optimization v0.9").
- **Cryptographic Proof:** Each completed task generates a unique "Hash" or "Signature". This serves as the "proof of work" the user can show to employers. "I own the signature for this contribution."

## 3. Ownership: "High-Command" Projects
**Goal:** Allow users to own hard, chunkable problems.
**Concept:** Distinguish between "Grunt Work" (micro-tasks) and "Command Projects" (hard problems).
**Feature:**
- **Project Ownership:** Users can "Claim" a complex problem. Once claimed, it's theirs for a set time (e.g., 48 hours).
- **Attribution:** When a major project is completed, the "Why Am I Doing This" lineage for *future* tasks explicitly names the architect. (e.g., "Based on the protocol established by [User X]").

## Proposed Implementation Steps
1. **Build the Dossier Page:** A profile view listing completed tasks as "Verified Artifacts" with unique signatures.
2. **Update Dashboard:** Add the "Global Operation" status bar to visualize the collective drumbeat.
